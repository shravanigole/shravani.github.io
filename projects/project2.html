Hotel Website

Website | HTML, CSS, JavaScript | 2024

About the Project

This is a clean and modern hotel website designed to display rooms, features, and amenities. For bookings, users can directly reach out via the contact number provided.

Key Features

  • Responsive and modern UI for hotel details
  • Dedicated sections for room types and amenities
  • Contact section with phone number for booking
  • Gallery section with images of the hotel

My Role

As the sole developer and designer of this project, I:

  • Designed the entire UI using HTML and CSS from scratch
  • Used JavaScript to enhance interactivity (like room filtering, scroll effects, etc.)
  • Ensured responsive design for all devices
  • Added contact form (linked to email or WhatsApp) instead of online payment for simplicity

Tech Stack
